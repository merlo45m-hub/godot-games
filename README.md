# Agent 007 — Stealth FPS

A first-person stealth FPS built in Godot 4 for Android.

## Setup

1. Open in Godot 4.4+ editor
2. Set main scene to `scenes/main.tscn`
3. Export for Android via `project → export`

## CI Build

Push to `main` — GitHub Actions builds an APK automatically.
Download from the workflow artifacts.

## Controls

| Input | Action |
|-------|--------|
| Left stick / WASD | Move |
| Right stick / Mouse | Look |
| Fire button / Click | Shoot |
| Sprint button / Shift | Sprint |
| Crouch button / Ctrl | Crouch |
| Reload button / R | Reload |
| Escape | Pause / Unlock mouse |
